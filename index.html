Sobre o Projeto

Recentemente, os Correios alteraram o sistema de endereçamento de Paraíso do Tocantins, substituindo o CEP geral por códigos específicos para cada logradouro.

Essa mudança gerou diversos desafios logísticos, erros de entrega e dúvidas entre os moradores. Diante deste cenário, este projeto surgiu como uma individual e sem fins lucrativos.

O objetivo é oferecer uma ferramenta simples e rápida para auxiliar a comunidade a encontrar e copiar seus novos códigos postais, mitigando as frustrações causadas pela transição.

Renato Uchôa é desenvolvedor e, nas horas livres, gosta de contribuir com a comunidade, utilizando a tecnologia para criar ferramentas que tragam soluções práticas e facilitem o dia a dia das pessoas.
